## Getting Started

Run the development server:

```
npm run dev
# or
yarn dev
```

Run the production server:

```
npm run build
# or
yarn build
```

Run esLint Script: 
```
npm run lint
# or
yarn lint
```


Run prettier Script: 
```
npm run format
# or
yarn format
```
